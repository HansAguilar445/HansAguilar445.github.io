HTML and CSS

Hypertext Markup Language (HTML) is a markup language that web developers build websites with. Consisting of several elements and tags to structure the page, these elements remain within either the head or body of the page. Cascading Style Sheets (CSS) are files made in the language of the same name that websites used to decorate and change the structure of the site.

Tags and Elements

HTML consists of various tags designed to become a structured page when viewed in a browser. These tags serve different functions depending on where they are placed in a site. The following list details specific tags that developers must use for a functional website. HTML consists of various tags designed to become a structured page when viewed in a browser.

  • <!DOCTYPE html> ensures that browsers can identify the type of file they are opening.

  • <html> contains all other tags and elements.

  • <head> contains data that is invisible to a user without opening the developer tools. Within it, you would use <meta> tags to define specific information about the site like keywords or how the site conforms to a userâ€™s screen.

  • <body> contains tags that become visual elements when the file is opened through a web browser. This is the part that users see and the part that CSS affects.

Within the body, the tags form visual elements that behave differently depending on what they are. While CSS can change their behaviour, these elements can be classified as being an inline element or a block element by default. Block elements form a new line when they appear, while inline elements remain on the same line as the previous elements unless something forces them to appear underneath the previous line like a line break. Examples of block elements include paragraphs, headings, and tables. Examples of inline elements include <span> tags, anchors, and <br> tags.

Attributes

Within HTML tags, attributes define specific behaviours. They are always stated in the starting tag and often are formatted as attribute="value". Examples of attributes include links to other pages, alternate text for an image, or the language of the page.

CSS Rules, Selectors, and Values

In a CSS file, a rule defines how HTML elements react. These rules consist of selectors and declarations of specific behaviours. The selectors are usually the names of the tags being targeted, an example being a <div>. A declaration consists of a CSS property and a value, an example being display: flex.
