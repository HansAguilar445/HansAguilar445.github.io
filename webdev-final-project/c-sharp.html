The Basics of C#

C# is an object-oriented programming language developed by Microsoft using their .NET framework. It is very similar to the Java programming language, having been initially designed as such to conform to ECMA’s standards. Applications that are programmed in C# include:

  • Games built in the Unity Engine

  • The Nintendo Switch emulator called Ryujinx

  • Paint.NET

C# was first released in 2002 with the .NET Framework’s 1.0 release. Its most recent version is C# 10.0, released in November of 2021 alongside .NET 6.0.

Setting up

Before you start programming, please ensure the following conditions have been fulfilled:

  • Your IDE must be compatible with .cs files

  • .NET must be installed and up to date

Once these conditions have been fulfilled, you can start by creating a file. For the purpose of this guide, you should use Visual Studio as the IDE as it does not require much setup. The file type for C# code is .cs, though Visual Studio also adds .sln files to contain the project.

Basic rules of programming

Now that you have a file to write a program in, there are specific conventions that you must remember when writing code in C#. These are shared with most other programming languages, so keep the following rules in mind when following this beginner’s guide:

  • Always end lines of code with semicolons (;). This tells the computer to go to the next line when you run the code, so forgetting one can break the entire program.

  • Remember that code is case-sensitive. Not capitalizing one method that you want to use when it needs a capital letter is another mistake that causes errors and breaks the program.

  • Name your variables something that makes sense for what they do. It helps you figure out what purpose every variable fulfills if something goes wrong after running the program.

  • Indent your code. Code that is misaligned is hard to read.

  • Put comments in your code. This helps you define what role each chunk of code serves in the overall program. Use // for comments that use one line or /* and */ to define where a multi-line comment starts and ends.

Console outputs

To start out simply, you will write some sentences in the terminal. It is the traditional starting point for any beginner in a programming language, the classic "Hello World" message. The template for a console application always starts with the message, but you might not understand how it all works. To explain, the message is sent to the terminal using either the method called WriteLine() or a similar method called Write(), which are methods connected to the Console object (Console.). You put data in the brackets, like Boolean (true or false) statements, equations, and strings of text, and it outputs the results. Some examples include:

  • Console.WriteLine("Hello World"); results in the words “Hello World” (quotation marks not included) being outputted from the console.

  • Console.WriteLine(1 > 0); results in a value of True being outputted to the console.

  • Console.WriteLine(2 * 6); results in a value of 12 being outputted to the console.

With this in mind, experiment to see what results you get from the terminal as practice before moving on.

Changing declared variables

Another important concept to keep in mind with variables is that you can change them later in the program. This is used in several ways, like iterating through loops and accepting user input. You can change the value of your variables by:

  • Defining a new value. Example: x = 5;
