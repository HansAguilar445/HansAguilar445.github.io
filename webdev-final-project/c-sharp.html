The Basics of C#

C# is an object-oriented programming language developed by Microsoft using their .NET framework. It is very similar to the Java programming language, having been initially designed as such to conform to ECMA’s standards. Applications that are programmed in C# include:

  • Games built in the Unity Engine

  • The Nintendo Switch emulator called Ryujinx

  • Paint.NET

C# was first released in 2002 with the .NET Framework’s 1.0 release. Its most recent version is C# 10.0, released in November of 2021 alongside .NET 6.0.

Setting up

Before you start programming, please ensure the following conditions have been fulfilled:

  • Your IDE must be compatible with .cs files

  • .NET must be installed and up to date

Once these conditions have been fulfilled, you can start by creating a file. For the purpose of this guide, you should use Visual Studio as the IDE as it does not require much setup. The file type for C# code is .cs, though Visual Studio also adds .sln files to contain the project.

Basic rules of programming

Now that you have a file to write a program in, there are specific conventions that you must remember when writing code in C#. These are shared with most other programming languages, so keep the following rules in mind when following this beginner’s guide:

  • Always end lines of code with semicolons (;). This tells the computer to go to the next line when you run the code, so forgetting one can cause errors and break the program.

  • Remember that code is case-sensitive. Not capitalizing one method that you want to use when it needs a capital letter is another mistake that causes errors and breaks the program.

  • Name your variables something that makes sense for what they do. It helps you figure out what purpose every variable fulfills if something goes wrong after running the program.

  • Indent your code. Code that is misaligned is hard to read.

  • Put comments in your code. This helps you define what role each chunk of code serves in the overall program. Use // for comments that use one line or /* and */ to define where a multi-line comment starts and ends.

Console outputs

To start out simply, you will write some sentences in the terminal. It is the traditional starting point for any beginner in a programming language, the classic "Hello World" message. The template for a console application always starts with the message, but you might not understand how it all works. To explain, the message is sent to the terminal using either the method called WriteLine() or a similar method called Write(), which are methods connected to the Console object (Console.). You put data in the brackets, like Boolean (true or false) statements, equations, and strings of text, and it outputs the results. Some examples include:

  • Console.WriteLine("Hello World");, which results in the words "Hello World" (quotation marks not included) being outputted from the console.

  • Console.WriteLine(1 > 0);, which results in a value of True being outputted to the console.

  • Console.WriteLine(2 * 6);, which results in a value of 12 being outputted to the console.

With this in mind, experiment to see what results you get from the terminal as practice. The code from before will work, so copy it and adjust the values to see what happens. Once you feel ready, continue to the next part.

As the name implies, these are placeholders that you define to substitute for actual data. There are seven main data types that variables can be defined as, which the following table will detail:

In order to use variables, you need to declare them using the data type’s name and define them with a value. These are formatted as datatype variableName = value;. From there, you can use them as you need by using the name to stand in for the value in the code you are writing.

Changing declared variables

Another important concept to keep in mind with variables is that you can change them later in the program. This is used in several ways, like iterating through loops and accepting user input. You can change the value of your variables by:

  • Defining a new value. Example: x = 5;

  • Using specific operators. Examples: x++;, x--;, x *= 3;

  • Using type-specific methods to define a new value. Example: str = str.Replace(oldString, newString);

Be careful not to change the variables in ways that they cannot accept, such as adding a string to an int, as that can cause errors.

Conditional Statements

With variables explained, conditional statements are next for important concepts. These allow you to control what actions the computer takes when specific conditions are fulfilled, like when a variable exceeds a certain size. There are two main types of conditional statements, the "if-else" type and the "switch-case" type.

If-else statements

The "if-else" statement type is the most common type of conditional statement that you use. You use these when you want to set one or more conditions for code to trigger, especially if you have multiple variables you want to use. Take the following steps to use one:

  1. Write if and put a set of round brackets next to it. It should look like this: if ()

  2. In the round brackets, write the conditions you want for the code to execute. Examples: x == 1, x > 0, x < 2, x == true (do not use this as it is redundant)

    • You can also set more than one if you need to using OR (||) or AND (&&) between conditions. Example: (x > str.Length && !active)

  3. After the round brackets, add a set of curly braces. They can be right next to the start of the statement or on the next line. Example: if (x > 0) {}

  4. In between the curly braces, write the code you want to trigger when the if statement is fulfilled.

In the end, it should look like this:

if (condition)

{

//Put your code here

}

You can also follow up by using else if (condition) and/or else to trigger when your first set of conditions are not met and another set is met. The following code block demonstrates this:

//The second option is picked

int x = 10;

if (x < 10)

{

Console.WriteLine("x is less than 10.");

} else if (x == 10)

{

Console.WriteLine("x is 10.");

} else

{

Console.WriteLine("x is greater than 10.");

}

Switch-case statements

The "switch-case" statement type is the other main type of conditional statement that you can use in your code. It gives you an easier time with setting up multiple conditions in certain situations and it is easier to read. You would use these when you want to open up more options for your program when one specific variable is the deciding factor for its path.

To create a "switch-case" statement like the one above, follow these steps:

  1. Write switch() and put the name of the controlling variable in the brackets. Example: switch(str)

  2. Put a set of curly braces next to or underneath the start of the statement.

  3. In between the curly braces, write the options. They should be formatted as case value:. Example,: case "monday":, case "friday":

    • You can also set a default option. To set a default option, place default: at the end of your statement.

  4. After each option, write the code that you want to execute underneath each case.

  5. At the end of the code you want to execute for each case, add break;. This will exit the statement once the code under each case has run.

The following code block is a "switch-case" statement in use to track days of the week:

//Triggers the Sunday option

string str = "sunday";

Console.WriteLine("What day is it?")

switch(str)

{

case "monday":

Console.WriteLine("Today is Monday.");

break;

case "tuesday":

Console.WriteLine("Today is Tuesday.");

break;

case "wednesday":

Console.WriteLine("Today is Wednesday.");

break;

case "thursday":

Console.WriteLine("Today is Thursday.");

break;

case "friday":

Console.WriteLine("Today is Friday.");

break;

case "saturday":

Console.WriteLine("Today is Saturday.");

break;

case "sunday":

Console.WriteLine("Today is Sunday.");

break;

default:

Console.WriteLine("Not a day.");

break;

}

Keep in mind that unless you want to put "if-else" statements inside the cases, only one variable can be set at a time to control the statement. This is not the case with "if-else" statements, where you can set several conditions using multiple variables. Use these when you only have one main variable to control how the code functions.

Loops

Now that conditional statements are out of the way, the final main concept a C# programmer will need to remember is the concept of looping code. Loops allow you to reduce the number of lines you need to write by repeating specific lines that you place inside a loop. There are three main kinds of loops that you normally use in C#.
